# React Dev Finder

A GitHub user search app built with React and Vite. Look up any GitHub user by username to view their profile information, repositories, followers, and more.

## Features

- Search GitHub users by username
- View profile details: avatar, bio, location, and join date
- See public repository count, followers, and following
- Links to user's blog and social profiles
- Responsive design for desktop and mobile

## Tech Stack

- React
- Vite
- GitHub REST API
- JavaScript (ES6+)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/jerichoNega/react-dev-finder.git

# Navigate into the project
cd react-dev-finder

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

## Build

```bash
npm run build
```
